Composio MCP Integration Guide

Integrating with **Composio MCP** is incredibly simple and can be done in just five steps:

1. Visit the [Composio MCP Directory ](/309f261a6dfe80418eb1e11810b99f66)page 
1. Ensure you are logged in, else signup (its free)
1. In the dashboard head to Install Section
![image](https://raw.githubusercontent.com/sunilcomposio/notion-to-github/main/composio-mcp-integration-guide/images/1.png)
1. Select Cursor from the list click on Install.
![image](https://raw.githubusercontent.com/sunilcomposio/notion-to-github/main/composio-mcp-integration-guide/images/2.png)
1. In the redirected page, Click on “Install in Cursor”
![image](https://raw.githubusercontent.com/sunilcomposio/notion-to-github/main/composio-mcp-integration-guide/images/3.png)
1. You will be redirected to Cursor, and in the MCP server Page, click Install
![image](https://raw.githubusercontent.com/sunilcomposio/notion-to-github/main/composio-mcp-integration-guide/images/4.png)
1. Once installed, click on Connect, theb Authenticate at redirect, and done!
![image](https://raw.githubusercontent.com/sunilcomposio/notion-to-github/main/composio-mcp-integration-guide/images/5.png)
